# 🍽️ MERN Food Delivery Service

A full-stack food delivery web application built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js). This project simulates a real-world food ordering and delivery service, featuring user authentication, dynamic menus, cart functionality, and order tracking.

---

## 🚀 Features

- 👨‍🍳 Browse food items by category
- 🛒 Add to cart and manage orders
- 🔐 User authentication (Login/Register)
- 🧾 Order history & tracking
- 💬 Contact/Feedback form
- 📦 Admin panel to manage inventory (WIP)
- 🖥️ Responsive frontend using React

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS / CSS Modules (optional)
- Axios for API calls

**Backend:**
- Node.js
- Express.js
- MongoDB (via Mongoose)

**Other:**
- JWT Authentication
- RESTful APIs
